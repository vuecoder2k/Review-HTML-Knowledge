Basic Concepts of HTML

What's HTML?

HTML Là từ viết tắt của Hyper Text Markup Language, sử dụng để tạo các siêu văn bản gồm có text, image, sound, video, link, ...

HTML được sử dụng để xây dựng cấu trúc của một trang web.

HTML cung cấp các tags (thẻ) để xây dựng các thành phần của trang web.

What's HTML tags

Tag (thẻ) trong HTML có cấu trúc như sau:

<tag> ... content ... </tag>

<tag>: thẻ này được gọi là thẻ mở (open tag)

</tag>: thẻ này được gọi là thẻ đóng (close tag)

... content ...: chứa nội dung text hoặc có thể là những thẻ HTML khác.

Lưu ý: Thẻ đóng (close tag) rất quan trọng, khi bạn code lưu ý phải luôn có thẻ đóng.

Mỗi thẻ trong HTML đều có nhiệm vụ chức năng riêng: <h1> hiển thị tiêu đề chính trong HTML document, <h2> hiển thị tiêu đề phụ thứ 2, <img> hiển thị hình ảnh vào trong HTML document, ...

Lưu ý: việc sử dụng các thẻ HTML vào đúng chức năng nhiệm vụ của nó rất quan trọng, giúp search engine và sreen reader hiểu nội dung cấu trúc trang web tốt hơn từ đó trang web dễ dàng lên SEO hơn.

Types of tags

Trong HTML có 2 loại thẻ HTML:

  • Content tags

  • Empty tags

Content tags: Là những thẻ HTML có chứa nội dung trong thẻ có thẻ đóng và thẻ mở.

Empty tags: Là những thẻ HTML không có thẻ đóng, không có nội dung chỉ có mỗi thẻ mở

What's HTML elements

Mọi tags (thẻ) HTML đều được gọi là một HTML elements

How an elements display on browser?

HTML elements có 2 cách hiển thị khác nhau trên browser:

  • Inline

  • Block

Dạng hiển thị inline: mỗi element sẽ nằm cạnh element khác trên cùng một dòng

Dạng hiển thị block: mỗi element sẽ luôn nằm ở dòng mới.

What's HTML attributes

Các attributes được khai báo ở thẻ ở cấu trúc attributes được khai báo theo cặp giá trị key=value, với key là tên attribute và value là giá trị của attribute đó.

Mỗi elements đều có những attributes (thuộc tính) giúp bổ sung thêm các thông tin cho element đó, thay vì để nó như mặc định.

Attributes là tính chất như: màu sắc, kích cỡ, vị trí, ... của một element

Mỗi elements có rất nhiều attributes khác nhau

Mặc định một số HTML elements bắt buộc phải đi với một số attributes.

Một element có thể có những attributes riêng mà các elements khác không có.

What's <!DOCTYPE html> ?

Thành phần này chịu trách nhiệm khai báo cho browser biết rằng đây là một file document HTML theo chuẩn HTML 5.

What's <html>?

Đây được gọi là thẻ HTML root, mọi thẻ HTML khác muốn được browswer xử lý đều phải nằm trong thẻ này.

Browser sẽ dựa vào thẻ này để render nội dung trang web.

What's <head> ?

Đây là nơi hiển thị tên trang web, khai báo các tài nguyên, mô tả nội dung trang web, các metadate phục vụ trang web.

What's <body> ?

Nơi hiển thị lên trang web người dùng nhìn thấy

Chủ yếu 90% chúng ta sử học cách sử dụng các thẻ trong body.

HTML tags in head

Trong thẻ head gồm có các thẻ HTML sau:

  • <title>: Hiển thị nội dung HTML document trên thanh tab của browser

  • <meta>: sử dụng để khai báo, mô tả một số thông tin cần thiết cho trang web

  • <link>: Nhúng một cài tài nguyên bên ngoài vào trang web

  • <script>: Dùng để chứa mã JavaScript

  • <style>: Dùng để chứa mã CSS

  • <base>: Khai báo vị trí mở các liên kết tài nguyên bên ngoài.

HTML form

Thẻ form tạo ra các ô nhập liệu khác nhau giúp người dùng tương tác hoặc gửi dữ liệu lên trang web.

Cấu trúc thẻ form như sau: 


<form>

... form elements

</form>

Form elements gồm các thẻ mà <form> hỗ trợ để tạo mọi các trường input hoặc nhãn thông báo cho user biết và tương tác với web.

<form> elements gồm các thẻ sau:

  • <input> - Là element phổ biến nhất trong form, input element hỗ trợ tạo nhiều trường, ô dữ liệu khác nhau thông qua attributes type cho user tương tác với web

  • <label> - Tạo nhãn hướng dẫn, hỗ trợ user click vào các ô, trường dữ liệu trong <form>.

  • <textarea> - Tạo trường nhập text cho phép user nhập nhiều dòng text

  • <select> - Tạo drop down menu cho phép user select các item trong thẻ <option>

  • <option> - Định nghĩa từng item trong thẻ <select>, <datalist> và thẻ <optgroup>

  • <button> - Tạo một nút bấm

  • <datalist> - Tạo một trường nhập các items được định nghĩa trước trong thẻ <option>

  • <output> - Hiển thị kết quả của một thẻ input nào đó

  • <fieldset> - Sử dụng để nhóm một nhóm các trường, ô input có liên quan vào một nhóm

  • <legend> - Tạo một caption lên trên <fieldset>

  • <optgroup> - Sử dụng để nhóm một nhóm các item liên quan trong thẻ <option> và gán nhãn cho từng nhóm items

<form> attributes

Chúng ta sẽ cùng tìm hiểu một số attributes cơ bản nhưng quan trọng của <form>:

  • action - chỉ định nơi server sẽ xử lý dữ liệu gửi lên

  • method - Chỉ định phương thức truyền tải dữ liệu lên server (GET/POST)

    GET
    - Mục đích: chỉ phù hợp để lấy dữ liệu từ server
    - Cách truyền: dữ liệu được truyền trên thanh URL của browser theo cặp giá trị key=value, và bị giới hạn độ dài dữ liệu, và user có thể nhìn thấy các dữ liệu được truyền
    - Lưu ý: Không truyền những dữ liệu nhạy cảm theo method này
    POST
    - Mục đích: sử dụng để gửi dữ liệu lên xử lý server
    - Cách truyền: dữ liệu được truyền trong HTTP body an toàn hơn khi gửi các dữ liệu nhạy cảm, method này user thường sẽ không nhìn thấy dữ liệu.
    - Lưu ý: method này không bảo mật dữ liệu, nó chỉ là không cho user thường nhìn thấy dữ liệu. Muốn bảo mật phải thêm HTTPS.

  • target - chỉ định vị trí nơi dữ liệu trả về sẽ hiển thị tại đâu

  • novalidate - Tắt định kiểm tra định dạng đầu vào các input trong <form>

  • autocomplete - Có 2 trang thái cho attribute này on/off. Nếu là on thì các các input nhập liệu sẽ tự động lưu các giá trị nhập trước đó, ngược lại thì sẽ là off
