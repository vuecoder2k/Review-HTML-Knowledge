Basic Concepts of HTML

What's HTML?

HTML Là từ viết tắt của Hyper Text Markup Language, sử dụng để tạo các siêu văn bản gồm có text, image, sound, video, link, ...

HTML được sử dụng để xây dựng cấu trúc của một trang web.

HTML cung cấp các tags (thẻ) để xây dựng các thành phần của trang web.

What's HTML elements

HTML elements: mỗi một tag (thẻ) HTML đều được gọi là một HTML elements

Mỗi elements trong HTML đều có nhiệm vụ chức năng riêng: <h1> hiển thị tiêu đề chính trong HTML document, <h2> hiển thị tiêu đề phụ thứ 2, <img> hiển thị hình ảnh vào trong HTML document, ...

Mỗi elements đều có những attributes (thuộc tính) giúp bổ sung thêm các thông tin cho element đó, thay vì để nó như mặc định.

Lưu ý: việc sử dụng các HTML elements vào đúng chức năng nhiệm vụ của nó rất quan trọng, giúp search engine hiểu và sreen reader hiểu nội dung cấu trúc trang web tốt hơn từ đó trang web dễ dàng lên SEO hơn.

Types of elements

Trong HTML có 2 loại emlements:

  • Content elements

  • Empty elements

Content elements: Là những thẻ HTML có chứa nội dung trong thẻ có thẻ đóng và thẻ mở.

Empty elements: Là những thẻ HTML không có thẻ đóng, không có nội dung.

How an elements display on browser

HTML elements có 2 cách hiển thị khác nhau trên browser:

  • Inline

  • Block
